Плагин для браузера Hat который позволяет выполнять автоматизированное тестирование данных в базе данных MySql
Скачать плагин: https://gitflic.ru/project/somovstudio/hatpluginmysql

Системные требования:
Плагин совместим с браузером Hat начиная с версии 1.4.6
Скачать по ссылке: https://gitflic.ru/project/somovstudio/hat

Официальная страница приложения: https://somovstudio.github.io/hat.html
Документация online: https://somovstudio.github.io/help/Hat/index.html
